Display Utilities Reference

Class Description
d-none Hide element (display: none)
d-block Block display
d-inline Inline display
d-inline-block Inline-block display
d-flex Flexbox display
d-grid CSS Grid display
d-*-none, d-*-block, etc. Responsive display (sm, md, lg, xl, xxl)

Responsive Breakpoints

Display utilities can be applied at different breakpoints:

  • d-* - Extra small (all screens)
  • d-sm-* - Small (≥576px)
  • d-md-* - Medium (≥768px)
  • d-lg-* - Large (≥992px)
  • d-xl-* - Extra Large (≥1200px)
  • d-xxl-* - 2XL (≥1400px)
Tip: Resize your browser to see the responsive display changes in action!
