Accordion Components

Component Description
.accordion Container for accordion items
.accordion-item Individual accordion item
.accordion-header Clickable header button
.accordion-button Button that controls collapse/expand
.accordion-collapse Content area that toggles
.accordion-body Text content within collapsed area
data-bs-parent Only one item open at a time
