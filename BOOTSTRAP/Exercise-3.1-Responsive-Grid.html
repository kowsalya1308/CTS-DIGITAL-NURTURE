Three columns that stack on mobile, 2 per row on tablets, 3 per row on desktops.

How It Works

On Mobile (< 768px)

Columns stack vertically (full width) because no breakpoint prefix is applied.

col-md-6 col-lg-4
→ Without col-12, columns take full width

On Tablet (≥ 768px)

Columns arrange in 2 per row (col-md-6 = 50% width each).

col-md-6 = 50% width (2 columns per row)

On Desktop (≥ 992px)

Columns arrange in 3 per row (col-lg-4 = 33.33% width each).

col-lg-4 = 33.33% width (3 columns per row)

Grid Classes Used

  • container - Fixed width responsive container
  • row - Creates flex row
  • col-md-6 - 50% on medium+ screens
  • col-lg-4 - 33.33% on large+ screens
Tip: Resize your browser or open on different devices to see the responsive behavior in action!
