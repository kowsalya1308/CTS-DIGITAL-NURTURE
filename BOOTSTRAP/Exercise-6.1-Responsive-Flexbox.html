Flexbox Utilities Reference

Class Description
d-flex Display flex (create flex container)
flex-row Flex direction: row (default, left to right)
flex-column Flex direction: column (top to bottom)
flex-md-row Switch to row on medium+ screens
flex-md-column Switch to column on medium+ screens
justify-content-start Align items to start (default)
justify-content-center Center items horizontally
justify-content-between Distribute items evenly
align-items-center Center items vertically
gap-2, gap-3, gap-4 Add spacing between flex items
Tip: Resize your browser to see the responsive behavior. On mobile (< 768px), elements stack vertically (flex-column). On desktop (≥ 768px), they arrange in a row (flex-md-row).
