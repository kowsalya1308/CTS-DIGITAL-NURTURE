How Floating Labels Work

  • Wrap input and label in .form-floating
  • Label must come AFTER the input
  • Input must have a placeholder attribute
  • Label floats above input when focused or filled
  • Works with text, email, password, select, textarea
Note: Floating labels require Bootstrap 5.1+. They provide a modern, space-efficient form design.
