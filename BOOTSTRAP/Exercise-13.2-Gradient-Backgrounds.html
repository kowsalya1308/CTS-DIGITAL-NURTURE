Gradient Backgrounds Reference

Utility Description
bg-gradient Adds gradient background effect (works with any bg-* color)
bg-gradient bg-primary Creates a blue gradient background
bg-gradient bg-dark text-white Dark gradient with white text (good contrast)
text-white-50 Semi-transparent white text for hierarchy
Tip: Combine bg-gradient with any background color class and add text-white for better contrast and modern appearance.
