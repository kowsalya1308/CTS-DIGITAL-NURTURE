Exercise 1.2: Bootstrap 5 Local Setup

This page uses locally downloaded Bootstrap 5 files.

Note: This setup requires downloading Bootstrap 5 files and placing them in a bootstrap folder.

Project Structure

Expected folder structure:

project-folder/
├── index.html
├── bootstrap/
│   ├── css/
│   │   ├── bootstrap.min.css
│   │   └── bootstrap.css
│   ├── js/
│   │   ├── bootstrap.bundle.min.js
│   │   └── bootstrap.bundle.js
│   └── (source files if using Sass)
└── js/
    └── main.js (optional)

Advantages of Local Setup

  • Works offline
  • Faster loading from local disk
  • Easy to customize Sass files
  • No CDN dependency

Installation Steps

  1. Download Bootstrap from getbootstrap.com
  2. Extract to your project folder
  3. Link CSS and JS files in HTML
  4. Include bootstrap.bundle.min.js for plugins
