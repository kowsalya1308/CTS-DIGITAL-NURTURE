Exercise 5.2: Column Reordering

Using order-md-1, order-md-2, etc. to reorder columns on different screen sizes

Example 1: Basic Reordering

On Desktop: Column 2 appears first, Column 1 second
On Mobile: Columns appear in natural order (HTML order)
Column 1 - order-md-2 (Second on desktop)
Column 2 - order-md-1 (First on desktop)

Example 2: Three Column Reorder

Desktop Order: 2 → 3 → 1
Mobile Order: 1 → 2 → 3 (natural order)
Column 1 - order-md-3
Column 2 - order-md-1 (First)
Column 3 - order-md-2 (Second)

Example 3: Different Reordering at Different Breakpoints

Mobile: 1 → 2 → 3
Tablet (md): 3 → 1 → 2
Desktop (lg): 2 → 3 → 1
Column 1
order-md-2 order-lg-2
Column 2
order-md-3 order-lg-3
Column 3
order-md-1 order-lg-1

Practical Example: Responsive Layout

Sidebar on right on mobile, left on desktop:

Main Content - order-md-1
Sidebar - order-md-2

Order Utilities Reference

Class Description
order-1, order-2, ..., order-12 Set order for all screen sizes
order-sm-1, order-sm-2, ... Set order on small screens (≥576px)
order-md-1, order-md-2, ... Set order on medium screens (≥768px)
order-lg-1, order-lg-2, ... Set order on large screens (≥992px)
order-xl-1, order-xl-2, ... Set order on extra-large screens (≥1200px)
order-xxl-1, order-xxl-2, ... Set order on 2xl screens (≥1400px)
order-first Set order to -1 (appears first)
order-last Set order to 13 (appears last)
Tip: Order values range from 1-12. Resize your browser to see the reordering in action!
