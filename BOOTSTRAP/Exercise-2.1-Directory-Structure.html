Exercise 2.1: Bootstrap Directory Structure

Understanding the Bootstrap 5 file organization.

Bootstrap 5 Directory Structure

bootstrap-5.x.x/
├── dist/ (Compiled CSS and JS)
│ ├── css/
│ │ ├── bootstrap.css
│ │ ├── bootstrap.min.css
│ │ └── bootstrap.rtl.min.css (RTL support)
│ └── js/
│ ├── bootstrap.js
│ ├── bootstrap.bundle.js
│ └── bootstrap.bundle.min.js
├── scss/ (Sass source files)
│ ├── _variables.scss
│ ├── _functions.scss
│ ├── _mixins.scss
│ ├── _root.scss
│ ├── bootstrap.scss
│ └── (component SCSS files)
├── js/ (JavaScript source files)
│ ├── alert.js
│ ├── button.js
│ ├── carousel.js
│ └── (other component JS)
└── docs/ (Documentation)

Key Folder Purposes

dist/ Folder

Contains compiled and minified CSS and JavaScript files ready for production use.

Use these in production websites.
scss/ Folder

Contains Sass source files for customization and compilation.

Customize variables and recompile Bootstrap.
js/ Folder

Contains JavaScript component source files (requires bundler).

For advanced customization with build tools.
docs/ Folder

Contains the Bootstrap documentation website source.

Reference for component usage and examples.
Tip: For most projects, you only need the dist/ folder files. Use scss/ only if customizing Bootstrap colors and variables.
