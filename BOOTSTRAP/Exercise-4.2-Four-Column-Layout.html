Exercise 4.2: Four Column Equal Width Layout

Using col-sm-3 for 4 equal columns on small screens and above

Layout Info:
  • col-sm-3 = 25% width (4 columns per row)
  • Applies on small (≥576px) screens and above
  • On mobile (< 576px), columns stack vertically
