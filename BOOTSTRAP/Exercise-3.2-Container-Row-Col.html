Exercise 3.2: Container, Row, and Col Classes

Understanding the three fundamental Bootstrap grid classes.

1. Container Class

.container - Fixed width responsive container with max-width at each breakpoint

Max widths: 540px (sm), 720px (md), 960px (lg), 1140px (xl), 1320px (xxl)

Alternatives:
  • .container-fluid - 100% width always
  • .container-{breakpoint} - Responsive containers (container-md, container-lg, etc.)

2. Row Class

.row - Creates a flex container with -12px gutters by default

Must wrap column elements and provides alignment control.

Gutter classes: g-0, g-1, g-2, g-3, g-4, g-5

3. Column Classes (.col-*)

.col-* - Defines column widths at different breakpoints

Breakpoints: col (all), col-sm, col-md, col-lg, col-xl, col-xxl

Width values: 1-12 (e.g., col-md-6 = 50%, col-lg-4 = 33.33%)
