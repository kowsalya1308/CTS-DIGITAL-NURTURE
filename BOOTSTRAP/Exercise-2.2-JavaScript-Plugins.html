Exercise 2.2: Bootstrap JavaScript Plugins

Using bootstrap.bundle.min.js for interactive components.

Available JavaScript Plugins

  • Alert - Dismiss alert messages
  • Button - Toggle button states
  • Carousel - Image slider
  • Collapse - Expandable content
  • Dropdown - Dropdown menus
  • Modal - Dialog boxes
  • Offcanvas - Side panels
  • Popover - Popup tooltips
  • Scrollspy - Active link tracking
  • Tab - Tab navigation
  • Toast - Notifications
  • Tooltip - Hover tips

Bundle vs Individual Files

bootstrap.bundle.min.js

✓ Includes all Bootstrap plugins

✓ Includes Popper.js (for dropdowns/tooltips)

✓ Recommended for most projects

✓ No external dependencies needed

~87 KB minified
