Spacing Utilities Legend

  • m-* = Margin all sides
  • mt-*, mb-*, ms-*, me-* = Margin top, bottom, left, right
  • mx-*, my-* = Margin horizontal, vertical
  • p-* = Padding all sides
  • pt-*, pb-*, ps-*, pe-* = Padding top, bottom, left, right
  • px-*, py-* = Padding horizontal, vertical
  • ms-auto, me-auto = Push element left or right
